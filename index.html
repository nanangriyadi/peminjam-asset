📋 Cara Deploy Apps Script

  1. Buka Google Sheets kamu → Extensions → Apps Script
  2. Paste kode dari file Code.gs yang sudah disediakan
  3. Klik Deploy → New deployment
  4. Pilih type: Web App
  5. Execute as: Me (email kamu)
  6. Who has access: Anyone
  7. Klik Deploy → copy URL yang muncul
  8. Paste URL di kolom di atas → Simpan
